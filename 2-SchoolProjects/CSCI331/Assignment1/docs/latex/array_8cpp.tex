\doxylink{class_array}{Array} class:\+ like an int array (retains all functionality) but also includes additional features:\+ -- allows input and output of the whole array -- allows for comparison of 2 arrays, element by element -- allows for assignment of 2 arrays -- size is part of the class (so no longer needs to be passed) -- includes range checking, program terminates for out-\/of-\/bound subscripts

Assumptions:\+ -- size defaults to a fixed size of 10 if size is not specified -- array elements are initialized to zero -- user must enter valid integers when using \texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>} -- in \texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<}, integers are displayed 10 per line
