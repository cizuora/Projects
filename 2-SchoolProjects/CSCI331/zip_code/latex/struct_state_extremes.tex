\doxysubsection{Detailed Description}
Holds the extreme ZIP codes for a single state. 

\begin{DoxyPrecond}{Precondition}
All fields must be properly initialized before use 
\end{DoxyPrecond}
\begin{DoxyPostcond}{Postcondition}
Instances of \doxylink{struct_state_extremes}{State\+Extremes} can be used to track and update extreme ZIP codes for a state
\end{DoxyPostcond}
\begin{DoxyNote}{Note}
This structure maintains the four geographic extreme ZIP codes for a state:\+ 

easternmost, westernmost, northernmost, and southernmost. It updates these 

values as records are processed. 
\end{DoxyNote}
